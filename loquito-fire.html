Loquito Fire

Nota: Este es un prototipo que intenta probar una idea de un plataformero arcade y puede contener bugs. Se realizó para el concurso 3MM Pulmonear.

Sinópsis.

Un repentino fuego ataca el edificio que está siendo cuidado por nuestro héroe Tincho Caballero , un honrado conserje de día y científico de noche.

Salva todo el edificio apagando el fuego de cada nivel en menos de 3 minutos.

Controles

A / D = Movimiento izquierda / derecha
W / S = Usar escalera arriba / abajo
K = Saltar
J = Usar Matafuego
Esc = Pausa
Enter / J = Seleccionar
