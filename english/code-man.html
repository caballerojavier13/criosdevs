Code-Man

Save the computer, save the world.

Sinopsys

“Code-man” is a puzzle/adventure game where patience and skill is needed to complete each level. In total, there’re twenty four levels to go through and three skills (also called “add-ons”) that can be unlocked as we keep on playing.

The game sets in the year XXXX where all the nuclear armament is managed be a central computer, which is infected by a virus that pretends to shoot missiles to achieve a worldwide catastrophe. However, at the same time, the computer released two versions of an emergency security system called “Code-man”, which in turn are infected and corrupted. So, the last possibility is to run a third and ultimate manual version which you must use to prevent the apocalypses.
